# Fare Website
Fare (fà·re) meaning “to do” in Italian, is a prototype I made using React.

<b>Website:</b> [Fare](https://bitvivaz.com/fare/)

### Tech Stack 
- HTML
- CSS
- JavaScript
- ReactJS
- Ant Design
- Heroku
- NodeJS
- MongoDB

### Features
- Responsive Design
- Add, Edit and Delete Todocards
- Add, Edit and Delete Subtasks
